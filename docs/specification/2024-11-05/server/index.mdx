服务器通过 MCP 为语言模型提供添加上下文的基本构建模块。这些原语支持客户端、服务器和语言模型之间的丰富交互：

- **提示**：预定义的模板或指令，用于指导语言模型的交互
- **资源**：为模型提供额外上下文的结构化数据或内容
- **工具**：允许模型执行操作或检索信息的可执行函数

每个原语可以按照以下控制层级进行总结：

| 原语       | 控制方式             | 描述                                               | 示例                           |
|-----------|---------------------|---------------------------------------------------|-------------------------------|
| 提示       | 用户控制             | 用户选择调用的交互式模板                           | 斜杠命令、菜单选项             |
| 资源       | 应用程序控制         | 客户端附加和管理的上下文数据                       | 文件内容、Git 历史             |
| 工具       | 模型控制             | 暴露给语言模型以执行操作的函数                     | API POST 请求、文件写入         |
